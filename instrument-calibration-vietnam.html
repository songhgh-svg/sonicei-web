Accurate Instruments, Safe Systems

An uncalibrated instrument can cause incorrect process readings, missed alarms, and unsafe conditions at startup. We verify every instrument meets its specified accuracy before commissioning with documentation your PMC will accept.

// How We Work

5-Step Engagement Process

01
RECEIVE PROJECT SCOPE

Client shares scope of work number of loops, instrument tags, cable schedule, and timeline. We review within 24 hours and confirm feasibility. No retainer required to start discussions.

📋 Proposal within 24h · No obligation
02
REVIEW DRAWINGS & I/O LIST

We review P&IDs, loop diagrams, instrument index, I/O list, and cable schedule before mobilisation identifying gaps, ambiguities, and risks early. Issues raised to client before engineers go onsite, not after.

📐 P&IDs · Loop Diagrams · Instrument Index · Cable Schedule
03
DEPLOY ENGINEERS ONSITE

Core team mobilises to site Vũng Tàu / HCMC within 3–5 business days from PO. Offshore vessel deployment available from Vũng Tàu. We don't over-book; if we can't commit to your schedule, we say so upfront.

📍 Vũng Tàu · HCMC · Offshore · 3–5 days mobilisation
04
EXECUTE LOOP CHECK & TESTING

Systematic execution cable IR/continuity, instrument calibration, end-to-end loop verification, functional testing. Every defect raised as a punch item via AI-Punch™ with photo evidence. Client receives live punch list updates throughout.

⚡ AI-Punch™ · Live punch list · Photo evidence per item
05
HANDOVER WITH DOCUMENTATION

Complete turnover package signed loop check sheets, calibration certificates, punch list closeout (all A-items closed), as-built mark-ups, and PSSR sign-off. Delivered in client's preferred format. PMC acceptance facilitated.

📄 Loop Check Dossier · Calibration Certs · Punch List Closure · PSSR
